## Java 调用：真的只是一行代码的事

现在进入最关键的步骤——**手把手教你用 Java 调用 Gemini API**。假设你之前用 OpenAI 写过代码，现在想换成 Gemini，操作如下：

### 2. 替换 API Key
API Key 换成你从千聚ai后台申请的 API Key。

### 3. 指定模型名称
在请求参数里，把 `model` 字段改成 Gemini 的模型名。例如：

### 4. 完整 Java 示例代码
如果你用的是 OpenAPI 的 Java SDK（比如 `openai-java`），你的代码应该是这样：

java
import com.theokanning.openai.service.OpenAiService;
import com.theokanning.openai.completion.chat.*;

public class GeminiInvokeExample {
    public static void main(String[] args) {
        // 1. 设置千聚ai的 base URL
        String baseUrl = "https://www.qianjuai.com/v1";
        // 2. 设置你在千聚ai申请的 API Key
        String apiKey = "sk-你的千聚API_Key";
        
        // 3. 创建服务实例
        OpenAiService service = new OpenAiService(apiKey,  Duration.ofSeconds(60));
        // 注意：如果你的 SDK 没有直接支持自定义 baseUrl，需要构建 HttpClient 时指定
        // 这里简写思路，实际代码需根据你用的库自定义
        
        // 4. 构建对话消息
        ChatCompletionRequest completionRequest = ChatCompletionRequest.builder()
                .model("gemini-2.5-pro-exp-03-25")  // 指定 Gemini 模型
                .messages(List.of(
                        new ChatMessage(ChatMessageRole.USER.value(), "用 Java 写一个回调函数实现的防抖工具类")
                ))
                .build();
        
        // 5. 发起请求
        service.createChatCompletion(completionRequest)
                .getChoices()
                .forEach(choice -> System.out.println(choice.getMessage().getContent()));
    }
}


上面只是一个简写示例，实际项目中你需要根据你使用的 Java HTTP 客户端（如 OkHttp、HttpClient）和 OpenAI SDK 来适配 base URL。在 `openai-java` 库中，你可以用以下方式强制指定 base URL：
