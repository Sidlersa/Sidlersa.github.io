## Java 接入 Claude Sonnet 的核心代码

假设你已经在[千聚api中转站](https://www.qianjuai.com/)注册并拿到了API Key。你要做的只有三件事：指定模型名、设置正确的请求地址、发一个标准HTTP请求。下面是一个完整的Java类，依赖只需要Java 11+自带的`java.net.http`包和一个JSON处理库（比如Gson或Jackson）。我用Gson演示，因为最轻量。

java
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;
import com.google.gson.JsonArray;
import com.google.gson.JsonObject;

public class ClaudeSonnetSample {
    public static void main(String[] args) throws Exception {
        // 1. 从千聚获取的API Key
        String apiKey = "sk-你的千聚密钥";
        // 2. 关键：将base_url改为千聚的API地址
        String baseUrl = "https://www.qianjuai.com/v1";
        // 3. 选择模型，Claude Sonnet最新版
        String model = "claude-3-5-sonnet-20241022";

        HttpClient client = HttpClient.newHttpClient();

        JsonObject body = new JsonObject();
        body.addProperty("model", model);
        body.addProperty("max_tokens", 1024);

        JsonArray messages = new JsonArray();
        JsonObject userMsg = new JsonObject();
        userMsg.addProperty("role", "user");
        userMsg.addProperty("content", "用Java写一个快速排序，要完整可运行");
        messages.add(userMsg);
        body.add("messages", messages);

        HttpRequest request = HttpRequest.newBuilder()
                .uri(URI.create(baseUrl + "/chat/completions"))
                .header("Authorization", "Bearer " + apiKey)
                .header("Content-Type", "application/json")
                .POST(HttpRequest.BodyPublishers.ofString(body.toString()))
                .build();

        HttpResponse<String> response = client.send(request, HttpResponse.BodyHandlers.ofString());
        System.out.println("状态码: " + response.statusCode());
        System.out.println("响应内容: " + response.body());
    }
}


这段代码跑起来后，你会看到Claude Sonnet返回的完整JSON，里面包含你想要的快排代码。中间不需要任何代理、不需要二次配置。
