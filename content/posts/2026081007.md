### 第四步：写代码实现Llama4调用

来，我们开始写核心代码。创建一个名为 `Llama4ChatDemo` 的Java类。

我标好了每一步的作用，你看着敲就行。

java
import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;
import com.theokanning.openai.service.OpenAiService;
import java.time.Duration;
import java.util.List;

public class Llama4ChatDemo {
    public static void main(String[] args) {
        // 读取API Key，建议从环境变量或配置文件读取
        String apiKey = System.getenv("QIANJU_API_KEY");
        if (apiKey == null || apiKey.isBlank()) {
            System.out.println("错误：请设置QIANJU_API_KEY环境变量");
            return;
        }

        // 关键在这里：千聚的base_url
        // 替换成官方提供的接口链接
        OpenAiService service = new OpenAiService(
            apiKey,
            Duration.ofSeconds(60)  // 超时时间建议设长一些，模型较大响应时间会长
        );

        // 构建您的聊天请求
        ChatCompletionRequest request = ChatCompletionRequest.builder()
            .model("llama-4")  // 模型名称：千聚平台支持的Llama4模型ID
            .messages(List.of(
                new ChatMessage("user", "你好，请用Java实现一个简单的Hello World打印程序，并对每行代码加上注释。")
            ))
            .maxTokens(2000)  // 控制最大输出
            .build();

        // 发送请求并处理响应
        service.createChatCompletion(request)
            .getChoices()
            .stream()
            .forEach(choice -> {
                System.out.println("Llama4的回复：");
                System.out.println(choice.getMessage().getContent());
            });

        // 重要：调用结束后务必关闭服务释放资源
        service.shutdownExecutor();
    }
}


你看，代码就是这么简单。模型名字我写的是 `"llama-4"`，但在千聚后台，你最好确认一下具体支持的模型ID。因为Llama目前有多个变体，比如 `llama-4-sonnet` 或者 `llama-4-scout`。以千聚的实际模型列表为准。

**避坑第三条：** 模型名称非常重要。你把名字写错了，平台不会认出这个模型，直接返回404或者模型不存在错误。所以先在千聚后台搜一下你要的模型，把准确的ID写进代码。

### 如果报错怎么办

新手最常见的几个报错场景，我给你列出来，方便你查：

1. **401 Unauthorized**：你八成没传Key，或者传错了。检查环境变量名是否正确，或者Key前后有没有多余的空格。

2. **404 Not Found**：请求路径不对。先检查base_url是不是填成了 `https://www.qianjuai.com`（少了 `/v1`），再检查模型名称有没有写错。

3. **连接超时 / 连接拒绝**：网络问题。千聚可以直接在国内访问，如果你还是连不上，检查你的VPN是不是关了（好多工具会自动开全局代理）。实际上千聚自己就能直连，你关掉代理反而更稳定。

4. **类加载报错 / 方法不存在**：你可能用了不同版本的SDK。赶快去检查 `pom.xml`，把openai sdk版本改回 `0.12.0`，然后刷新Maven重新导入。

### 额外福利：流式输出怎么玩

上面的代码是一次性拿完整回复。但如果你想让Llama4像打字机一样一个字一个字往外蹦（体验更好），可以做流式输出。

核心区别就是把 `createChatCompletion` 换成 `streamChatCompletion`，然后处理 `EventSource`。

java
service.streamChatCompletion(request)
    .doOnComplete(() -> System.out.println("\n---流式输出完成---"))
    .subscribe(
        choice -> System.out.print(choice.getChoices().get(0).getMessage().getContent())
    );


**避坑第五条：** 流式输出的时候，记得在线程里处理，不要阻塞主线程。很多人在新手里直接在主线程调 `subscribe`，结果IDEA开始转圈圈，以为是卡死了，其实是函数没返回。正确做法是单独开一线程执行订阅。
