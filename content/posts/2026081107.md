## 完整Maven依赖：一步到位

现在我们来上硬菜。Moonshot的API接入完兼容OpenAI的接口标准，所以你只需要在Java项目里引入一个简单的HTTP客户端依赖，就能快速上手。这里我推荐使用OkHttp（行业标准）配合FastJson。

### 第一步：构建Maven依赖

在你的`pom.xml`文件中，加入以下依赖：

xml
<!-- OkHttp 核心依赖 -->
<dependency>
    <groupId>com.squareup.okhttp3</groupId>
    <artifactId>okhttp</artifactId>
    <version>4.12.0</version>
</dependency>

<!-- FastJson 用于JSON解析 -->
<dependency>
    <groupId>com.alibaba</groupId>
    <artifactId>fastjson</artifactId>
    <version>2.0.46</version>
</dependency>


为什么选OkHttp？因为它成熟、稳定、社区好，完全能满足API调用的所有需求。如果你用的是Spring Boot，OkHttp也是一个很干净的依赖，不会污染你的classpath。

**注意：** 如果你项目已经引入了Apache HttpClient或Spring的RestTemplate，也完全可以无缝替代。Moonshot API只认HTTP协议，不挑客户端。

### 第二步：代码实战——Moonshot接入Java示例

下面是最核心的部分。我直接写了一个简单的Java方法，演示如何调用Moonshot的对话模型。代码里已经帮你注释好了每行是什么意思，方便你直接复制到项目里运行。

java
import okhttp3.*;
import com.alibaba.fastjson.JSONObject;

public class MoonshotApiDemo {

    // Moonshot API 的基础地址（已映射至国内直连节点）
    private static final String BASE_URL = "https://www.qianjuai.com/v1";

    // 你在[千聚ai聚合平台](https://www.qianjuai.com/)申请的API Key
    private static final String API_KEY = "你的API_KEY";

    public static void main(String[] args) throws Exception {
        // 构建请求体（以moonshot-v1-8k模型为例）
        JSONObject requestBody = new JSONObject();
        requestBody.put("model", "moonshot-v1-8k");
        
        JSONObject message = new JSONObject();
        message.put("role", "user");
        message.put("content", "用Java写一个冒泡排序算法。");
        
        JSONObject messages = new JSONObject();
        messages.put("messages", new JSONObject[]{message});
        // 合并请求体（注意这里把messages放到了外部）
        // 为了简化代码，采用标准格式：
        JSONObject fullBody = new JSONObject();
        fullBody.put("model", "moonshot-v1-8k");
        fullBody.put("messages", new JSONObject[]{message});
        fullBody.put("temperature", 0.7);
        fullBody.put("max_tokens", 1024);

        // 构建HTTP请求
        OkHttpClient client = new OkHttpClient.Builder()
                .connectTimeout(30, java.util.concurrent.TimeUnit.SECONDS)
                .readTimeout(30, java.util.concurrent.TimeUnit.SECONDS)
                .build();

        Request request = new Request.Builder()
                .url(BASE_URL + "/chat/completions")
                .addHeader("Authorization", "Bearer " + API_KEY)
                .addHeader("Content-Type", "application/json")
                .post(RequestBody.create(
                        MediaType.parse("application/json"),
                        fullBody.toJSONString()
                ))
                .build();

        // 发送请求并处理响应
        Response response = client.newCall(request).execute();
        if (response.isSuccessful()) {
            String responseBody = response.body().string();
            JSONObject jsonResponse = JSONObject.parseObject(responseBody);
            String content = jsonResponse.getJSONArray("choices")
                    .getJSONObject(0)
                    .getJSONObject("message")
                    .getString("content");
            System.out.println("Moonshot 回复: " + content);
        } else {
            System.err.println("请求失败: " + response.code() + " " + response.message());
        }
    }
}


**这段代码的运行流程：**
1. 调用`okhttp3`构建请求
2. 使用FastJson构造标准JSON请求体
3. 向`https://www.qianjuai.com/v1/chat/completions`发起POST请求
4. 接收AI回复并打印
