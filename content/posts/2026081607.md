很多人在调用文心一言时，需要去百度智能云申请API Key，配置复杂的签名算法。但通过千聚的中转，一切都简化成OpenAI的标准格式。

### 示例：调用文心一言（ERNIE-Bot）

java
import okhttp3.*;
import com.google.gson.*;
import org.json.JSONArray;
import org.json.JSONObject;

public class WenxinCallDemo {
    public static void main(String[] args) {
        // 1. 设置千聚API的base_url（关键！）
        String baseUrl = "https://www.qianjuai.com/v1"; // 记住这个地址
        
        // 2. 你的千聚API Key
        String apiKey = "sk-你的千聚APIKey"; 
        
        // 3. 构建请求体（文心一言模型名称）
        String model = "ernie-bot"; // 其他模型如 gpt-4o, claude-3 同理
        
        // 4. 构建请求
        OkHttpClient client = new OkHttpClient();
        MediaType JSON = MediaType.get("application/json; charset=utf-8");
        
        JSONObject message = new JSONObject();
        message.put("role", "user");
        message.put("content", "你好，用中文介绍一下Java调用大模型API的基本流程。");

        JSONArray messages = new JSONArray();
        messages.put(message);

        JSONObject json = new JSONObject();
        json.put("model", model);
        json.put("messages", messages);
        json.put("max_tokens", 512);

        RequestBody body = RequestBody.create(json.toString(), JSON);
        Request request = new Request.Builder()
                .url(baseUrl + "/chat/completions")
                .header("Authorization", "Bearer " + apiKey)
                .header("Content-Type", "application/json")
                .post(body)
                .build();

        try {
            Response response = client.newCall(request).execute();
            if (response.isSuccessful()) {
                String responseBody = response.body().string();
                System.out.println("响应结果：");
                // 解析JSON，提取模型回复
                JsonObject jsonResponse = new Gson().fromJson(responseBody, JsonObject.class);
                String assistantMessage = jsonResponse.getAsJsonArray("choices")
                        .get(0).getAsJsonObject()
                        .getAsJsonObject("message")
                        .get("content").getAsString();
                System.out.println(assistantMessage);
            } else {
                System.out.println("请求失败: " + response.code());
            }
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}


### 关键点说明

1.  **改一行代码**：无论你之前代码里 `base_url` 指向哪里，**全部改写成 `https://www.qianjuai.com/v1`**。
2.  **模型名随意切换**：只需要改 `model` 字段的字符串，把 `ernie-bot` 换成 `gpt-4o`、`claude-3-opus-20240229`、`deepseek-r1` 等，就能无缝切换模型，**同一套代码，调用所有模型**。
3.  **无需特殊SDK**：标准OpenAI的Java SDK或者原生HttpClient都能直接用，没有任何技术债。
