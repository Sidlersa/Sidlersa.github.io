然后新建一个 Java 类 `ClaudeTest.java`，把下面的代码贴进去：

java
import okhttp3.*;
import com.fasterxml.jackson.databind.JsonNode;
import com.fasterxml.jackson.databind.ObjectMapper;
import com.fasterxml.jackson.databind.node.ArrayNode;
import com.fasterxml.jackson.databind.node.ObjectNode;

import java.io.IOException;

public class ClaudeTest {
    private static final String API_URL = "https://www.qianjuai.com/v1/chat/completions";
    // 将下面的占位符替换为你在千聚API后台获取的Key
    private static final String API_KEY = "sk-your-api-key-from-qianjuai";

    public static void main(String[] args) {
        OkHttpClient client = new OkHttpClient.Builder()
                .connectTimeout(30, java.util.concurrent.TimeUnit.SECONDS)
                .readTimeout(60, java.util.concurrent.TimeUnit.SECONDS)
                .build();

        ObjectMapper mapper = new ObjectMapper();
        ObjectNode requestBody = mapper.createObjectNode();

        // 指定模型为 Claude Opus 4.1
        requestBody.put("model", "claude-opus-4.1");
        requestBody.put("max_tokens", 1024);

        // 构建消息列表
        ArrayNode messages = mapper.createArrayNode();

        ObjectNode userMessage = mapper.createObjectNode();
        userMessage.put("role", "user");
        userMessage.put("content", "你好，请用50字以内的中文介绍一下你自己。");
        messages.add(userMessage);

        requestBody.set("messages", messages);

        Request request = new Request.Builder()
                .url(API_URL)
                .header("Authorization", "Bearer " + API_KEY)
                .header("Content-Type", "application/json")
                .post(RequestBody.create(
                        requestBody.toString().getBytes(),
                        MediaType.parse("application/json")
                ))
                .build();

        try (Response response = client.newCall(request).execute()) {
            if (!response.isSuccessful()) {
                System.out.println("请求失败，HTTP状态码: " + response.code());
                System.out.println("响应内容: " + response.body().string());
                return;
            }
            String responseBody = response.body().string();
            JsonNode responseJson = mapper.readTree(responseBody);

            // 从标准OpenAI响应格式中提取助手的回复
            String assistantReply = responseJson
                    .get("choices")
                    .get(0)
                    .get("message")
                    .get("content")
                    .asText();

            System.out.println("Claude Opus 4.1 的回复: ");
            System.out.println(assistantReply);

        } catch (IOException e) {
            System.err.println("发生网络异常: " + e.getMessage());
            e.printStackTrace();
        }
    }
}


**关键点只有两个：**
1. 把 `API_URL` 设置成 `https://www.qianjuai.com/v1/chat/completions`。
2. `API_KEY` 替换成你自己的。

代码逻辑完全跟随标准 OpenAI Chat API 格式，唯一变的就是 `model` 字段写成了 `claude-opus-4.1`。
