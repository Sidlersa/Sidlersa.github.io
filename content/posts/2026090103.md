我用一个最简单的 Maven 项目做示例，演示如何用[千聚api聚合站](https://www.qianjuai.com/)调用 Claude 3.5 Sonnet。

### 2. 核心 Java 代码

java
import com.theokanning.openai.OpenAiService;
import com.theokanning.openai.completion.chat.*;
import java.util.*;

public class Claude3Test {
    public static void main(String[] args) {
        // 关键：只改以下两行
        String token = "sk-你的千聚apiapikey"; 
        String baseUrl = "https://www.qianjuai.com/v1";
        
        // 创建服务实例
        OpenAiService service = new OpenAiService(token, Duration.ofSeconds(60));
        service.setBaseUrl(baseUrl);  // 设置中转站地址
        
        // 构造消息
        List<ChatMessage> messages = new ArrayList<>();
        ChatMessage systemMsg = new ChatMessage("system", "你是Java技术专家，请用简洁准确的语言回答。");
        ChatMessage userMsg = new ChatMessage("user", "解释一下Java中Stream API的并行流工作原理。");
        messages.add(systemMsg);
        messages.add(userMsg);
        
        // 指定模型：这里用 claude-3.5-sonnet-20241022
        ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model("claude-3.5-sonnet-20241022")
                .messages(messages)
                .maxTokens(1024)
                .temperature(0.7)
                .build();
        
        // 发起调用
        service.createChatCompletion(request).getChoices().forEach(choice -> {
            System.out.println(choice.getMessage().getContent());
        });
    }
}


**就完了**。`model` 里直接写明确的 Claude 模型 ID，`baseUrl` 改成[千聚api聚合站](https://www.qianjuai.com/)的 API 地址，`token` 换成你在千聚申请的 Key，然后代码不用动任何一行。
