## 接入有多简单：Java 示例

Kimi K2 完全兼容 OpenAI API，所以你只需要把客户端 endpoint 指向千聚。下面是一段标准的 Java 代码，用 OkHttp 调用 Kimi K2 流式输出：

java
import okhttp3.*;
import org.json.JSONObject;
import java.io.*;

public class KimiK2Example {
    private static final String API_KEY = "你的千聚API密钥";
    private static final String BASE_URL = "https://www.qianjuai.com/v1";

    public static void main(String[] args) throws IOException {
        OkHttpClient client = new OkHttpClient.Builder()
                .addInterceptor(chain -> {
                    Request request = chain.request().newBuilder()
                            .addHeader("Authorization", "Bearer " + API_KEY)
                            .build();
                    return chain.proceed(request);
                })
                .build();

        JSONObject payload = new JSONObject();
        payload.put("model", "kimi-k2");  // 千聚支持的模型名
        payload.put("messages", new JSONArray("[{\"role\":\"user\",\"content\":\"你好，Kimi K2！\"}]"));
        payload.put("stream", true);

        Request request = new Request.Builder()
                .url(BASE_URL + "/chat/completions")
                .post(RequestBody.create(MediaType.parse("application/json"), payload.toString()))
                .build();

        try (Response response = client.newCall(request).execute()) {
            if (!response.isSuccessful()) throw new IOException("请求失败: " + response);
            BufferedReader reader = new BufferedReader(new InputStreamReader(response.body().byteStream()));
            String line;
            while ((line = reader.readLine()) != null) {
                if (line.startsWith("data:")) {
                    String data = line.substring(5).trim();
                    if (!"[DONE]".equals(data)) {
                        JSONObject json = new JSONObject(data);
                        String content = json.optJSONArray("choices")
                                .optJSONObject(0)
                                .optJSONObject("delta")
                                .optString("content", "");
                        System.out.print(content);
                    }
                }
            }
        }
    }
}


**改动点**：`BASE_URL` 换成千聚的地址，`API_KEY` 换成千聚申请的 key，`model` 换成千聚支持的 Kimi K2 模型名即可。其他代码结构完全不需要动。

如果你用的是 Spring Boot 或 HttpClient，思路一模一样：改 `baseUrl`，换 `apiKey`。
