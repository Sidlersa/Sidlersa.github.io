- **原因：** API Key 填写错误，或 Key 已过期。
- **解决：** 重新在千聚后台复制正确的 Key。注意，开头不要有空格。如果 Key 被盗用，立刻在后台重置。

**问题 2：`404 Not Found` 或 `Model Not Found`**
- **原因：** 你输入的模型名称不准确。豆包模型名是大小写敏感的，比如 `doubao-pro-128k` 不能写成 `Doubao-Pro-128k`。

**问题 3：`Content Filtered` 或请求被拒绝**
- **原因：** 你的提示词可能触发了豆包的内容安全过滤机制。
- **解决：** 调整提示词，使其更安全、合规。这属于大模型的安全策略，不是平台的问题。

**问题 4：响应速度极慢或经常断开**
- **原因：** 网络环境问题，或者你使用了不稳定的网络。
- **解决：** 确保你的服务器或本地网络可以稳定访问 `www.qianjuai.com`。如果使用公司网络或机场，尝试切换节点或更换网络。
