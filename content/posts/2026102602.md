### Java接入示例：完整代码与注释

我们用Spring Boot + OkHttp实现一个简单的图文对话客户端。首先在 `pom.xml` 添加依赖：

</dependency>
<dependency>
    <groupId>com.alibaba</groupId>

</dependency>


然后编写核心调用类：

java
import okhttp3.*;
import com.alibaba.fastjson.JSONObject;
import com.alibaba.fastjson.JSONArray;
import java.io.IOException;
import java.util.Base64;

public class QwenVlClient {
    private static final String BASE_URL = "https://www.qianjuai.com/v1";
    private static final String API_KEY = "sk-你的千聚API Key"; // 替换为你的Key
    private static final String MODEL = "qwen-vl-plus";

    private final OkHttpClient client;
    private final MediaType JSON = MediaType.get("application/json; charset=utf-8");

    public QwenVlClient() {
        this.client = new OkHttpClient.Builder()
                .connectTimeout(30, TimeUnit.SECONDS)
                .readTimeout(60, TimeUnit.SECONDS)
                .build();
    }

    /**
     * 发送图文混合对话请求
     * @param base64Image 图片的Base64编码（不含data:image部分）
     * @param text 用户文字问题
     * @return 模型回答内容
     */
    public String chatWithImage(String base64Image, String text) throws IOException {
        // 构建消息数组
        JSONObject userMessage = new JSONObject();
        userMessage.put("role", "user");
        JSONArray content = new JSONArray();

        // 文本部分
        JSONObject textPart = new JSONObject();
        textPart.put("type", "text");
        textPart.put("text", text);
        content.add(textPart);

        // 图片部分（指定为base64格式）
        JSONObject imagePart = new JSONObject();
        imagePart.put("type", "image_url");
        JSONObject imageUrl = new JSONObject();
        imageUrl.put("url", "data:image/png;base64," + base64Image);
        imagePart.put("image_url", imageUrl);
        content.add(imagePart);

        userMessage.put("content", content);

        JSONObject requestBody = new JSONObject();
        requestBody.put("model", MODEL);
        requestBody.put("messages", new JSONArray().add(userMessage));
        // 可选参数：temperature, max_tokens等
        requestBody.put("max_tokens", 1024);

        Request request = new Request.Builder()
                .url(BASE_URL + "/chat/completions")
                .header("Authorization", "Bearer " + API_KEY)
                .post(RequestBody.create(requestBody.toJSONString(), JSON))
                .build();

        try (Response response = client.newCall(request).execute()) {
            if (!response.isSuccessful()) {
                throw new IOException("Unexpected code " + response);
            }
            String responseBody = response.body().string();
            JSONObject json = JSONObject.parseObject(responseBody);
            // 解析返回的choices数组
            String contentReply = json.getJSONArray("choices")
                    .getJSONObject(0)
                    .getJSONObject("message")
                    .getString("content");
            return contentReply;
        }
    }

    // 快速测试
    public static void main(String[] args) throws IOException {
        QwenVlClient client = new QwenVlClient();
        // 假设有一张图片的Base64（实际请替换）
        String base64Img = "iVBORw0KGgoAAAANSUhEUg...";
        String answer = client.chatWithImage(base64Img, "请描述这张图片的内容");
        System.out.println(answer);
    }
}

- `BASE_URL` 指向千聚的地址，完全兼容OpenAI的 `/chat/completions` 端点。
- 图片使用 `data:image/png;base64,...` 格式，Qwen-VL支持直接传入。

### 图文教程：如何配置图片Base64

在实际项目中，通常从文件或URL读取图片，转为Base64字符串。这里给一个工具方法：

java
import java.io.FileInputStream;
import java.io.IOException;
import java.util.Base64;

public class ImageUtils {
    public static String imageToBase64(String filePath) throws IOException {
        FileInputStream fis = new FileInputStream(filePath);
        byte[] bytes = fis.readAllBytes();
        return Base64.getEncoder().encodeToString(bytes);
    }

    // 从URL获取图片并转Base64（依赖OkHttp等）
    public static String urlToBase64(String imageUrl, OkHttpClient client) throws IOException {
        Request req = new Request.Builder().url(imageUrl).build();
        Response resp = client.newCall(req).execute();
        byte[] bytes = resp.body().bytes();
        return Base64.getEncoder().encodeToString(bytes);
    }
}


调用时只需：
