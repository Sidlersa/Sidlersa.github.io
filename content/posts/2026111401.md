### 完整Java接入代码（这才是干货）

废话不多说，直接上代码。这绝对是你今天能看到最清晰、最直接的Java接入o3的示例。

这个示例使用OpenAI官方提供的Java客户端库，但只改了`baseUrl`这一行。复制粘贴，配置好你的API Key，就能立刻运行。

**第一步：在你的`pom.xml`中添加依赖**

xml
<dependency>

**第二步：完整的Java主程序**

java
import com.theokanning.openai.OpenAiService;
import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;
import com.theokanning.openai.completion.chat.ChatMessageRole;

import java.time.Duration;
import java.util.ArrayList;
import java.util.List;

public class QianjuO3Example {

    public static void main(String[] args) {
        // ==========================================
        // 魔法就在这一行：只需将baseUrl替换为千聚API的地址
        // ==========================================
        String baseUrl = "https://www.qianjuai.com/v1"; // 这是你的专属入口
        String apiKey = "你的千聚API Key";  // 去 https://www.qianjuai.com/register 注册获取

        // 创建OpenAiService实例，超时时间根据需求设置
        OpenAiService service = new OpenAiService(apiKey, Duration.ofSeconds(60));
        // 如果你是Spring项目，可以注入一个配置好的RestTemplate，这里使用默认的

        // 构建消息列表
        List<ChatMessage> messages = new ArrayList<>();
        messages.add(new ChatMessage(ChatMessageRole.USER.value(), "你好，请用中文简单介绍一下你自己，并说说你属于哪个系列的大模型。"));

        // 构建请求，指定模型为o3-mini
        ChatCompletionRequest completionRequest = ChatCompletionRequest.builder()
                .model("o3-mini") // 模型名，千聚支持：o3-mini, o3-mini-high, o1, o1-mini 等
                .messages(messages)
                .maxTokens(1000) // o3系列推荐设置合理的max_tokens
                .build();

        try {
            System.out.println("正在调用千聚API接入的o3-mini模型...");
            // 调用API
            String response = service.createChatCompletion(completionRequest)
                    .getChoices().get(0).getMessage().getContent();
            System.out.println("模型回复内容如下：");
            System.out.println("========================================");
            System.out.println(response);
            System.out.println("========================================");

        } catch (Exception e) {
            System.err.println("请求失败，请检查网络和API Key配置：" + e.getMessage());
        } finally {
            // 记得关闭service，释放资源
            service.shutdownExecutor();
        }
    }
}



**是不是就这么简单？** 你完全不需要修改其他任何逻辑。把代码里的`baseUrl`和`apiKey`换成你的，`model`换成你想测试的模型名（如`o3-mini-high`），就能无缝衔接o3的全部能力。
