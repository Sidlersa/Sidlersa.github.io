## 进阶功能：流式输出 & 工具调用

很多时候你需要模型一个字一个字输出，比如实现打字机效果。GLM-4.5 在千聚上也原生支持流式输出：

python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_QIANGU_API_KEY",
    base_url="https://www.qianjuai.com/v1"
)

stream = client.chat.completions.create(
    model="glm-4.5",
    messages=[{"role": "user", "content": "用中文解释量子纠缠。"}],
    stream=True  # 开启流式
)

for chunk in stream:
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="", flush=True)


如果你的应用需要让模型调用外部函数（比如搜天气、查数据库），GLM-4.5 支持 tool call，千聚也完全兼容标准接口：

python
tools = [
    {
        "type": "function",
        "function": {
            "name": "get_weather",
            "description": "获取指定城市的实时天气",
            "parameters": {
                "type": "object",
                "properties": {
                    "city": {"type": "string", "description": "城市名称，如北京"}
                },
                "required": ["city"]
            }
        }
    }
]

response = client.chat.completions.create(
    model="glm-4.5",
    messages=[{"role": "user", "content": "北京今天天气怎么样？"}],
    tools=tools,
    tool_choice="auto"
)

# 解析模型返回的函数调用
if response.choices[0].message.tool_calls:
    print(response.choices[0].message.tool_calls[0].function)
