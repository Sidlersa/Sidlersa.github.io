以下是接入[千聚api聚合平台](https://www.qianjuai.com/)的完整Java代码示例：

java
import com.theokanning.openai.OpenAiService;
import com.theokanning.openai.completion.chat.*;
import java.util.List;

public class TestQianJuAI {
    public static void main(String[] args) {
        // 关键只有这一行：把api_base_url替换成[千聚api聚合平台](https://www.qianjuai.com/)的地址
        String baseUrl = "https://www.qianjuai.com/v1";
        String apiKey = "这里是你在[千聚api聚合平台](https://www.qianjuai.com/)获取的KEY";

        // 创建一个OpenAiService实例
        OpenAiService service = new OpenAiService(apiKey, 60);
        service.setBaseUrl(baseUrl);

        // 构建Chat请求
        ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model("gpt-4o")
                .messages(List.of(
                        new ChatMessage(ChatMessageRole.SYSTEM.value(), "你是一个Java开发助手"),
                        new ChatMessage(ChatMessageRole.USER.value(), "请用Java代码输出一句问候语")
                ))
                .maxTokens(100)
                .build();

        // 发起请求并打印结果
        service.createChatCompletion(request)
                .getChoices()
                .forEach(choice -> System.out.println(choice.getMessage().getContent()));
    }
}

### 如果你更习惯用原生HttpClient

如果不想用第三方SDK，直接用Java的`HttpClient`也可以：

java
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;
import java.time.Duration;

public class TestQianJuAIVanilla {
    public static void main(String[] args) throws Exception {
        String apiKey = "这里是你的KEY";
        String requestBody = """
            {
                "model": "gpt-4o",
                "messages": [{"role": "user", "content": "你好"}],
                "max_tokens": 50
            }
            """;

        HttpClient client = HttpClient.newBuilder()
                .connectTimeout(Duration.ofSeconds(30))
                .build();

        HttpRequest request = HttpRequest.newBuilder()
                .uri(URI.create("https://www.qianjuai.com/v1/chat/completions"))
                .header("Content-Type", "application/json")
                .header("Authorization", "Bearer " + apiKey)
                .POST(HttpRequest.BodyPublishers.ofString(requestBody))
                .build();

        HttpResponse<String> response = client.send(request, HttpResponse.BodyHandlers.ofString());
        System.out.println("返回结果:" + response.body());
    }
}


运行后同样会拿到一个完整的JSON响应。解析一下`choices[0].message.content`字段就拿到了AI的回复。
