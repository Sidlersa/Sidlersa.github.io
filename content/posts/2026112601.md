## 完整代码模板——直接复制就能用

为了让你更省事，我把一个更完整的、可直接运行的代码模板放在下面。你只需要把里面的 `your_api_key_here` 换成你自己的API Key，保存为 `ai_writer.py` 文件，用Python运行就行。

python
import os
from openai import OpenAI

# 配置部分：从环境变量读取API Key（推荐），或直接写在这里
API_KEY = os.getenv("QIANJU_API_KEY", "你的千聚API Key")  # 替换 "你的千聚API Key"
BASE_URL = "https://www.qianjuai.com/v1"

# 初始化客户端
client = OpenAI(api_key=API_KEY, base_url=BASE_URL)

# 调用模型生成文章
def generate_article(prompt, model="gpt-4o"):
    try:
        response = client.chat.completions.create(
            model=model,
            messages=[
                {"role": "system", "content": "你是一位顶级的内容营销专家，擅长撰写有深度、高转化的文案。"},
                {"role": "user", "content": prompt}
            ],
            max_tokens=2048,  # 控制生成文章的长度
            temperature=0.7   # 控制创造性的高低：0-1之间，越大越有创意
        )
        return response.choices[0].message.content
    except Exception as e:
        return f"调用API时出错：{e}"

# 主程序
if __name__ == "__main__":
    user_prompt = "请帮我写一篇500字左右的软文，介绍AI技术如何帮助小型创业公司提升效率。"
    print("正在生成文章，请稍候...\n")
    article = generate_article(user_prompt)
    print(article)


这个模板包含了完整的错误处理、参数配置，你可以直接把它当作你第一个AI写作工具的基础组件。
